# MEAN Stack Basic Items List

## Description
This is a basic MEAN stack web application that fetches and displays a list of items from a MongoDB database.

The project uses:
- Angular for the frontend
- Node.js with Express.js for the backend API
- MongoDB for data storage

No authentication, advanced validation, or complex UI features are included.

---

## Features
- Fetch a list of items from MongoDB
- Display items in a simple Angular component
- RESTful API built with Express.js
- MongoDB collection for storing items

---

## Technologies Used
- Angular
- Node.js
- Express.js
- MongoDB
- Mongoose
